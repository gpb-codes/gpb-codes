## Sobre mí

> Construyo software donde la **arquitectura es intencional**, el **código es mantenible** y la **automatización elimina la fricción**.

Mi trabajo se ubica en la intersección de **ingeniería IA** y **desarrollo de productos full stack**. Actualmente construyendo en **Drakkar Labs** — un laboratorio de innovación que desarrolla herramientas combinando LLMs, pipelines RAG y infraestructura web moderna para resolver problemas reales de productividad y gestión del conocimiento.
